# MsiklmGUI

A gui for MSIKLM. It allows you to change the color and mode of your MSI laptop keyboard

## Requeriments
Msi laptop and [MSIKLM]

## Use
Open the .java with sudo, custom your settings and apply. Also, you can make,save and load your custom profiles
